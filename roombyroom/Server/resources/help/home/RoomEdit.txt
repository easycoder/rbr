## Room Editor

The Room Editor is where you configure each of your rooms. It starts with a menu, headed by the name of the room, where you can

  - Edit the room name - see the ~tid:RoomName~ Room Name editor
  - Edit the IP address of the temperature sensor - see the ~tid:Sensor~ Sensor IP editor
  - Edit the IP addresses of each of the relays controlled in this room - see the ~tid:Relays~ Relay IP editor
  - Edit the schedule for timed mode - see the ~tid:Timing~ Timing editor

~tid:RoomTools:Go to the Room Tools Help page~
