## Main Menu

~img:mainmenu.jpg:50% center~
This is the main system menu. Here you can set up a new system, add rooms, backup the configuration or restore it. You can also evaluate RBR without the need to buy any hardware or software.

~tid:MAC:Set the MAC address~

~tid:Password:Set the system password~

~tid:Name:Set the system name~

~tid:AddRoom:Add a room~

~tid:Backup:Back up your configuration~

~tid:Restore:Restore your configuration to the backed-up version~

~tid:Evaluate:Evaluate RBR~

~tid:content:Return to the initial help page~
